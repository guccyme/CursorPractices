# 多语言Hello World GUI应用

这是一个使用Python/Tkinter开发的简单GUI应用程序,可以用不同的语言显示"Hello World"问候语。

## 功能特点

- 支持5种不同语言的问候语显示:
  - 英语
  - 中文
  - 法语
  - 西班牙语
  - 日语
- 每种语言使用独特的字体和颜色组合
- 针对MacOS优化的GUI显示效果
- 简洁清晰的用户界面

## 系统要求

- Python 3.x
- tkinter (Python标准库)
- 操作系统: Windows/MacOS/Linux
